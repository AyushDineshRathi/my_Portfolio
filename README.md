**Ayush Rathi – Personal Portfolio**

This is my personal portfolio website — a place where you can explore my work, skills, achievements, experience, and the projects I am most proud of.I built this portfolio to represent myself professionally and create a clean, interactive way for people to know me better.

**About This Website**
-------------------------

This portfolio includes:

*   A modern, clean UI built with **React + Vite**
    
*   Detailed sections about my **skills, projects, experience, education, and achievements**
    
*   An interactive **chat assistant** that knows everything in my profile and answers questions about me
    
*   A fully working **contact form** powered by EmailJS, allowing anyone to directly reach out
    
*   Responsive, smooth design suitable for all devices
    

The goal was to create something that represents me well, feels personal, and showcases both my **tech skills** and **design sense**.
